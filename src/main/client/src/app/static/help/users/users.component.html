Users of the HPO

The following list provides an overview of the groups who are currently using HPO. If you notice an omission, please contact us. The HPO is also a key component of the Monarch Initiative.

  • NIH Undiagnosed disease program (UDP). See also this paper that describes the UDP and its use of HPO: Gall T, et al (2017) Defining Disease, Diagnosis, and Translational Medicine within a Homeostatic Perturbation Paradigm: The National Institutes of Health Undiagnosed Diseases Program Experience. Front Med (Lausanne) 4:62.
  • Orphanet. See HIPBI-RD (Harmonising phenomics information for a better interoperability in the rare disease field) for information about work to integrate ORDO (the Orphanet Rare Disease Ontology), HPO (the Human Phenotype Ontology) as well as PhenoTips software for the capture and sharing of structured phenotypic data for RD patients.
  • Genome Canada.
  • Mouse Genome Informatics (MGI)
  • Genomics England; 100,000 Genomes Project
  • ECARUCA
  • DECIPHER
  • Deciphering Developmental Disorders (DDD)
  • GWAS Central
  • GWASdb
  • International Rare Diseases Research Consortium (IRDiRC)
  • ClinGen
  • NCBI Genetic Testing Registry
  • RD-Connect. See this description of RD-CONNECT.
  • Gene2MP.
  • MedSeq.
  • NCBI ClinVar
  • Sequence Ontology (SO). See this description of the use of phenotype ontologies in human GVF files.
