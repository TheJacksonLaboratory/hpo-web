The HPO project has developed a number of tools for quality control and biocuration. We do not anticipate that these tools will be of interest to most users, but we have made the source-code open source for some of these tools that we think might interest other groups involved in medical biocuration or ontology analysis.

  • The HPO Browser displays detailed information about terms and the diseases and genes annotated to them.
  • PhenoteFX is used for annotating disease models with HPO terms
  • HpoAnnotQc is used for quality control of the HPO annotation files.
  • Loinc2HPO is a Java library to map LOINC-encoded test results to Human Phenotype Ontology.
