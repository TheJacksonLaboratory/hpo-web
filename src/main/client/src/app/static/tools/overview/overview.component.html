HPO-based tools

The HPO has been used in many different tools and algorithms for clinical diagnostics, phenotype-driven genomic diagnostics, translational bioinformatics, and more. These pages provide an overview of tools developed within the HPO project and the Monarch Initiative as well as tools from other groups. If you have a tool that you would like us to add to these pages, please contact us.


Clinical diagnostics

The HPO provides a computational foundation for algorithms that implement fuzzy, specificity-weighted phenotype matching for computational differential diagnostic support. Users enter a list of HPO terms that represent the phenotypic abnormalities (signs, symptoms, laboratory measurements, etc.) observed in a patient. The tools employ a variety of approaches and algorithms to find diseases that best match the list of HPO terms.

  • The Phenomizer

Exome/Genome diagnostics and research

We have developed a series of Java-based tools that form the foundation of algorithms intended to implement phenotype-driven prioritization of variants in Mendelian disease. As input, the tools take a VCF file from an exome or genome as well as a list of HPO terms representing the phenotypic abnormalities observed in the person being investigated.

  • Exomiser
  • Genomiser
  • PhenIX is intended for clinical diagnostics using a "Mendeliome" panel or exome. The PhenIX algorithm can be run within the Exomiser tool.

CNV diagnostics

Array CGH and related assays are commonly used as a screening test for indications such developmental delay and congenital malformations. These assays can detect copy number variants (deletions and duplications). Numerous CNVs are found in all individuals, and thus it can be difficult to determine whether any given CNV is causally related to a medical problem. HPO-based analysis allows one to assess whether the genes contained within a CNV are phenotypically related to the phenotypic abnormalities observed in a patient.
