Phenomizer

The Phenomizer is a web-based application. for clinical diagnostics in human genetics using semantic similarity searches in ontologies Köhler et al., AJHG, October 2009. A detailed manual can be found in the online app.

A new application (Orphamizer) is being prototyped that used data from Orphanet as well as an improvbed algorithm and a different set of diseases and phenotype-associations.
