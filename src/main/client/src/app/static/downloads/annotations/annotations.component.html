Download Annotations

The annotation files (e.g. phenotype_annotation.tab) are available from our Hudson/Jenkins page. The formats of the annotation file are explained here.

  • phenotype_annotation.tab: contains manual and semi-automated annotations created by the HPO-team. These are annotations of OMIM-, Orphanet-, and DECIPHER-entries.
  • phenotype_annotation_hpoteam.tab: contains annotations made explicitly and manually by the HPO-team (mostly referring to OMIM entries)
  • negative_phenotype_annotation.tab: contains negative annotations (i.e. a disease is NOT associated with this HPO-term)
  • Genes_to_phenotype.txt and phenotype_to_genes.txt: Genes are associated with phenotypes by listing all of the phenotypes associated with any disease that is caused by mutation in the gene. See the FAQ about these two files to read about the differences between these two files.
  • phenotype.hpoa: The HPO project is transitioning to a new, improved file format for disease annotations. The file can be obtained here. Please see the online documentation for details about the new format, which will become the standard format after the transition period.

We are preparing a software solution that transforms the HPO ontology and annotation data into a relational database. Please contact us if you are interested in testing this software.
