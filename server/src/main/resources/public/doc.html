Search

Broad searching our data

Search terms, diseases or genes

Use this to find anything in our data.

query Parameters
q
required
string
Example: q=arach

Could be one of identifier (HPO, OMIM, ORPHA), an entrez id, or text.

max
integer
Example: max=-1

an upper limit for docs to return. [-1 for all]

offset
integer

a page offset when max value is set

category
string
Enum: "terms" "genes" "diseases"
Example: category=terms

filter search down to a category

Responses

https://hpo.jax.org/api/hpo/search
http://hpo.jax.org/api/hpo/search

Anchor descendant search starting from the given hpo term id

Use this to allow a search of the top level terms like modifier or onset. Works well with autocomplete

query Parameters
s
required
string
Example: s=HP:0003674

A valid human phenotype ontology identifier

q
required
string
Example: q=juvenile

a search string to look for [empty string returns everything under anchor]

Responses

https://hpo.jax.org/api/hpo/search/descendants/
http://hpo.jax.org/api/hpo/search/descendants/
